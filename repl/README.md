# Repl

This is a script programming language with Repl environment, This has some features and the ability to execute external commands and use the output.

## Example

```
if 5 < 3 { 
    # > sudo shutdown
}
else {
    # > echo "hi"
    # comment
}
```

## Using

```
$ cargo run
```
